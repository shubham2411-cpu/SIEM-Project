# Centralized Log Monitoring and Threat Detection using ELK Stack

## 📖 Overview

This project demonstrates a centralized log monitoring and threat detection solution using the ELK Stack. Windows Event Logs are collected using Winlogbeat, processed through Logstash, stored in Elasticsearch, and prepared for security monitoring.

## 🛠️ Technologies Used

- Ubuntu Linux
- Elasticsearch
- Logstash
- Winlogbeat
- Windows 10 Virtual Machine

## 🎯 Objectives

- Centralize Windows Event Logs
- Process logs using Logstash
- Store logs in Elasticsearch
- Verify successful log ingestion
- Demonstrate a basic SOC workflow

## 🏗️ Architecture

Windows VM → Winlogbeat → Logstash → Elasticsearch → Kibana
